Ans-4: HTML entities are special characters that cannot be included in HTML documents using normal keyboard characters. For example, if we want to display a copyright symbol (Â©) in your HTML document, you cannot simply type it on your keyboard, as it will not be recognized by the browser. Instead, you need to use an HTML entity to represent the symbol. HTML entities are written using a special syntax that consists of an ampersand (&), followed by a code or name that represents the entity, and then a semicolon (;). For example, the HTML entity for the copyright symbol is ©. HTML entities can be used to represent a wide range of special characters, such as mathematical symbols, currency symbols, and foreign language characters.
